# TikTok Shop Imagens

Ferramenta web para baixar imagens de produtos e videos do TikTok Shop.

## Funcionalidades

- Cole o link de um produto ou video do TikTok
- Busca automatica de todas as imagens
- Galeria com visualizacao ampliada
- Download de imagens individuais
- Download de todas as imagens em arquivo ZIP
- Interface responsiva para celular e computador
- Suporte a links curtos (vt.tiktok.com, vm.tiktok.com)

## Como usar

1. Acesse o site
2. Cole o link do TikTok (video ou produto)
3. Clique em "Buscar imagens"
4. Baixe individualmente ou todas de uma vez

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- JSZip

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Deploy

O projeto esta configurado para deploy automatico na Vercel atraves do GitHub.
